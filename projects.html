Personal projects

A list of projects that I have created for myself.

Irvi slovarik (dictionary) {% include icon-link.html url="http://irvi-slovarik.ru" icon-class="icon-small" %}

NOTE: Project is on russian

Irvi-slovarik is a dictionary for keeping personal lists of words and theirs translations. While learning a language you permanently discover new words and here you can keep them and test yourself with them.

Something like duolingo, but basically much simpler.

Currently under development, not ready for use.

Swan {% include icon-github.html username=site.github_username repo="swan" icon-class="icon-small" %}

Often there are some config files that need to be shared between many projects and keeping them up to date in every project can be annoying. Usually it is done by hands copypasting them manually from project to project. Swan is a tool to help automate this process.

Distributed as a rubygem.

Dotfiles {% include icon-github.html username=site.github_username repo="dotfiles" icon-class="icon-small" %}

A collection of my dotfiles and other configuration, with included tool to set everything up and get updates in matter of seconds on every machine.
